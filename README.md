# Preethi L — Portfolio

Personal portfolio for Preethi L, Software Developer. Built with vanilla HTML, CSS, and JavaScript.

## Stack

- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Libraries:** GSAP (ScrollTrigger) for animations, Font Awesome for icons
- **Fonts:** Inter, DM Sans (Google Fonts)
- **Deploy:** Static; works with GitHub Pages, Netlify, or any static host

## Structure

| File       | Purpose                    |
|-----------|----------------------------|
| `index.html` | Structure and content      |
| `style.css`  | Layout, design system, responsive |
| `script.js`  | Navigation, animations, form handling |
| `sw.js`      | Service worker (PWA cache) |

## Run locally

Open `index.html` in a browser, or use a local server:

```bash
npx serve .
# or
python -m http.server 8000
```
